## Overview

This project is a responsive personal portfolio website developed as the final project for a Web Development course. The website showcases my education, technical skills, coursework, experience, and software development projects while providing visitors with a way to contact me.

The website was built using HTML, CSS, and JavaScript and demonstrates the concepts learned throughout the course, including semantic HTML, accessibility, responsive design, Flexbox, CSS Grid, JavaScript event handling, and deployment.

## Installation

1. Clone the repository

git clone git@github.com:tcin-cyber/myportfolio.git

2. Open the project folder.

3. Open index.html in your browser.

No additional software or dependencies are required.

---

## Deployment

This project is deployed using GitHub Pages.

1. Push the repository to GitHub.
2. Open Repository Settings.
3. Select Pages.
4. Choose the main branch.
5. Save the settings.
6. Access the generated GitHub Pages URL.
